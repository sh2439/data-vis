Weather Visualization

This page contains 3 Tableau dashboards from the weather dataset. The dataset contains daily U.S. weather measurements in 2017, provided by the NOAA Daily Global Historical Climatology Network.

1. Temperature Visualization

This dashboard contains 2 sheets. The first sheet shows the racing bar chart of the ranking of monthly average temperature from top 30 states. It covers temperature from Jan 2017 to Sep 2017.

The second sheet shows the weekly average maximum temperature and minimum temperature from each state. If you hover on the data, it also shows the temperature difference. Both of 2 sheets are animated.

2. Temperature Map

This dashboard contains the monthly average temperature from each state on the map. The darker the color is, the higher temperature the state has. The sheet is animated.

3. Precipitation and Snow Fall Visualization

This dashboard contains 2 sheet. The first sheet shows the monthly average precipitation and snow fall from each state. Both precipitation and snow fall are measured in inch.

The second sheet contains the racing bar chart of monthly average precipitation of top 30 states. It covers precipitation from Jan 2017 to Sep 2017. Both of them are animated.
